# Auto-enter-comments-C
自动输入C语言文件头注释和函数头注释。

样式如下
文件头注释：

/**
  ******************************************************************************
  * @file  
  * @author  Su DaoZhen
  * @version  V1.0.0 
  * @date  2019/10/26
  * @brief  
  ******************************************************************************
  * @attention
  * 
  * (C)COPYRIGHT Su DaoZhen
  ******************************************************************************
*/





/*******************(C) COPYRIGHT Su DaoZhen *********END OF FILE*********/

函数头注释：

/*********************************************
* @brief  
* @param  
* @retval 
**********************************************/

注意：使用前请将输入法切换成英文状态！
